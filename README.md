# TigerFlow 🐯 — Gestion pour entreprises de dératisation

Application de gestion des interventions, contrats, devis et factures pour les entreprises 3D (dératisation, désinsectisation, désinfection) en France.

## Page : Détail du contrat

**Desktop** : sidebar de navigation, fiche client, montant HT/TTC/TVA, durée, fréquence, renouvellement tacite avec date limite de résiliation, prochaine intervention mise en avant, timeline des visites avec technicien assigné, progression du contrat, documents PDF.

**Mobile (UX app native)** :
- Barre de navigation en bas avec bouton "Planifier" central surélevé
- Carte "Prochaine intervention" affichée en premier
- Actions secondaires (PDF, renouveler, modifier) dans un menu ⋯
- Onglets fonctionnels : Informations / Interventions / Documents / Historique
- Email et téléphone cliquables, zones tactiles ≥ 44 px

## Lancer en local

Ouvrir `index.html` dans un navigateur. Aucune dépendance, aucun build.

## Mettre en ligne (GitHub Pages)

1. Settings → Pages
2. Source : branche `main`, dossier `/ (root)`
3. La page est en ligne à `https://VOTRE-COMPTE.github.io/tigerflow/`
